Course Snapshot

  • Duration: 3.5 Months
  • Format: In-person, 3x/week
  • Fee: ₹24,000
  • Location: West Pitampura – Near Kohat Enclave Metro
  • Level: Beginner to Intermediate

Week-by-Week Curriculum

Month 1: Python & Web Fundamentals

  • Python basics, functions, OOP
  • Git & GitHub version control

Month 2: Backend & AI APIs
